## 🧠 Example

```c
#include <stdio.h>

// Function to greet user
void greet(char *name) {
    printf("Hello, %s!\n", name);
}

int main() {
    char name[] = "Himanshu";
    greet(name);
    return 0;
}
```

**Output:**
```
Hello, Himanshu!
```
