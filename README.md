## 🚀 How to Run

### ✅ Prerequisites

Make sure you have a **C compiler** installed on your system:

- 🪟 **Windows** → Install [MinGW](https://www.mingw-w64.org/) or [TDM-GCC](https://jmeubank.github.io/tdm-gcc/)
- 🐧 **Linux/Mac** → GCC is usually pre-installed. Install via:
  ```bash
  sudo apt install gcc    # Ubuntu/Debian
  brew install gcc        # macOS (Homebrew)
  ```

### ▶️ Compile & Run

```bash
# Step 1: Clone the repository
git clone https://github.com/Himanshu962005/C-Code-College..git

# Step 2: Navigate into the folder
cd C-Code-College

# Step 3: Compile any program
gcc filename.c -o output

# Step 4: Run the compiled program
./output          # Linux/Mac
output.exe        # Windows
```
