> A powerful, high-performance, general-purpose programming language used in systems, embedded software, and beyond.

---

## ✨ Features

- Statically typed, compiled language with predictable performance.
- Direct memory management via pointers and manual allocation.
- Portable across platforms — write once, compile anywhere.
- Procedural paradigm with structured programming support.
- Rich standard library for I/O, math, strings, and more.

---

## ⚙️ Uses

- Build operating systems, kernels, and device drivers.
- Develop embedded systems and microcontroller firmware.
- Create compilers, interpreters, and language runtimes.
- High-performance scientific computing and simulations.

---

## 🧠 Example

```c
// Hello World in C
#include <stdio.h>

int main() {
    char name[] = "Himanshu";
    printf("Hello, %s!\n", name);
    return 0;
}
```
